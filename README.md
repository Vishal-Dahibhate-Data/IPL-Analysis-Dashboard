# 🏏 IPL Analysis Dashboard (2008–2025)

## 📌 Project Overview

The IPL Analysis Dashboard is an interactive Power BI solution designed to provide comprehensive insights into the Indian Premier League (IPL) from 2008 to 2025. The dashboard enables users to explore season-wise performance, team statistics, player achievements, points tables, and tournament trends through dynamic visualizations and interactive filters.

## 🎯 Objective

The objective of this project is to analyze IPL historical data and transform it into meaningful business insights using Power BI, helping users understand team performance, player contributions, and season outcomes.

## 📊 Dashboard Features

### Season Analysis
- Season-wise filtering (2008–2025)
- Champion and Runner-Up identification
- Tournament summary statistics

### Team Performance
- Team-wise points table
- Matches played, wins, losses, and points
- Comparative team analysis

### Player Statistics
- Orange Cap Holder Analysis
- Purple Cap Holder Analysis
- Most Fours Analysis
- Most Sixes Analysis
- Centuries and Half-Centuries Tracking

### Key Performance Indicators (KPIs)
- Total Matches
- Total Fours
- Total Sixes
- Total Venues
- Total Centuries
- Total Half-Centuries

## 🛠️ Tools & Technologies Used

- Power BI Desktop
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling
- Data Visualization
- Excel / CSV Dataset

## 📈 Business Insights Generated

- Identify the most successful teams across seasons.
- Analyze top-performing batsmen and bowlers.
- Compare season-wise team standings.
- Track batting milestones such as centuries, fours, and sixes.
- Evaluate tournament trends and performance patterns.
